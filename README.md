# CS2 Map Veto Simulator 🎮

A simple Python app that simulates CS2 map veto (like FACEIT) against an AI.

## Features

* BO1 / BO3 modes
* Random AI bans
* Map categories (Competitive, Wingman, Casual)
* Custom maps
* Map stats (% played)

## How to run

```bash
python cs2_map_veto_simulator.py
```

## Requirements

* Python 3.x (tkinter included)
